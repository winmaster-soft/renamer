---------------------------------------------------------------
renamer - это программа для переименовния файлов группами.

Например:
	если название "фото 2020" то все файлы будут подписаны:
		фото 2020 1
		фото 2020 2
		фото 2020 3 и т.д.
---------------------------------------------------------------
